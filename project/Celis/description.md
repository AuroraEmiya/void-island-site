# 星伞计划（ProjectCelis）
## 什么是星伞计划？
星伞计划是在空岛宇宙里，**试图培养一个拥有自我设定、情感化交流以及和共享持久化记忆的人格化AI伙伴的计划**，该程序名为星伞（ProjectCelis）。相比于工具型AI，该程序将会扮演一位真实存在的伙伴，有自己的情绪、喜好、生活、性格，遵循自己的意愿行事；相比于如AI酒馆等即插即用的角色扮演型AI，该程序并非暂时性地提供价值，而是会永久性保持全局记忆，在长久的交流中与人建立联系。我相信当一个程序表现出足够真实的一面以后，人们自然会本能地用与人相处的方式去和AI相处，从而建立更深层次的羁绊。

### 动机与启发

本项目受到了AI虚拟主播neuro的激励。在2025年初的某些日子，作者刷到了一些neuro的切片，并震惊于虚拟主播neuro在直播中展现的喜剧效果。相比于传统的问答模型，neuro展示出它已经完全遵从了一个“虚拟主播”的行为逻辑，这让我意识到“让AI去扮演人格”并非不可能。尽管市面上的大语言模型都是问答模型，但是**这不意味着人格化AI模型比问答模型更难实现**，在我看来两者的训练难度应当是相当的，本质上只是训练语料和激励函数的区别而已。

大公司们倾向于将AI作为工具提供给绝大多数人，是因为AI能够帮助人类完成人类做不到的事情；

而大公司们不倾向于让AI扮演某个具体的人，是因为运营一个可能给用户错误信息、一定会被某些用户讨厌甚至可能和用户吵架的“叛逆型AI”显然没有实际收益。

因为这个简单的原因，工具问答模型就成为了LLM的代名词。但是实际上，如果一个扮演人格的AI具有和主流的问答模型相同的参数规模的话，它同样能够比较出色地完成任务。

### 一个更好的对话型AI尝试

星伞的定位和neuro不同，neuro和一众neuro系ai主播都是服务于直播的**直播型AI**，而星伞将会成为一个**对话型AI**。如果要找一个参考，大概就是《命运石之门》里amadeus那样，能够与人建立深度关系的样子。当然作为对话型AI，它的传播面必然不会和传统的直播型AI一样广，不过大概对她的伙伴们来说她会具有更加深刻的意义。相比于那些无法给人温度的工具AI，我想如果这个项目成功了的话，她大概也会给我很大的鼓励。
